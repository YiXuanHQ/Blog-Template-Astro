## 📝 环境准备

### Redis安装

**推荐版本**：
- Redis 7.0（最新版）
- Redis 6.2（稳定版）

**安装方式**：
- Linux源码编译
- Docker容器
- Windows版本

### Java客户端

**推荐客户端**：
- **Jedis**：传统客户端
- **Lettuce**：异步客户端
- **Spring Data Redis**：Spring集成
