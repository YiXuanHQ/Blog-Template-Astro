## 🎯 原子化服务

### 什么是原子化服务

无需安装即可使用的轻量级应用形态。

### 开发原子化服务

```json5
// module.json5
{
  "module": {
    "type": "entry",
    "installationFree": true  // 标记为原子化服务
  }
}
```

### 原子化服务特点

```
优势
├─ 即点即用
├─ 占用空间小
├─ 更新便捷
└─ 推广容易
```
